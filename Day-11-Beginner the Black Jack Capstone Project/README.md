# Day 11: Blackjack Game

This is my project for Day 11 of the 100 Days of Python challenge by Angela Yu. The project is a text-based Blackjack game built in Python.

## Features
- Unlimited deck size
- Jack, Queen, King count as 10
- Ace counts as 1 or 11
- User vs Computer
- User can draw more cards or pass
- Automatic computer drawing until score >= 17
- Winner is decided based on Blackjack rules

## How to Play
1. Run `blackjack.py`
2. Follow on-screen instructions to draw or pass
3. Game automatically calculates scores and declares the winner
4. Option to restart the game after each round

## Technologies Used
- Python 3
- `random` module
- Functions, loops, conditionals
