# 100 Days of Python - Day 14: Higher or Lower Game

Welcome to Day 14 of my 100 Days of Python journey! In this project, I built the **Higher or Lower Game**, where the player guesses which account has more followers.

---

## üìù Project Description

The **Higher or Lower Game** is a simple interactive game where:
- Two accounts are shown on the screen with some information (name, description, and country).
- The player must guess which account has more followers on Instagram.
- If the player guesses correctly, their score increases by 1 and the next round begins.
- If the guess is wrong, the game ends and the final score is displayed.

The goal is to see how many correct guesses the player can make consecutively.
